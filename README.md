# MarketPlace

Vue.js frontend of a marketplace. Using Bootstrap for mobile-first and Vuex store for state management.

Mock data used in store to simulate data.

## Installation Versions
1. <b>npm 6.11.3</b>
2. <b>Vue.js 2.6.10</b>
3. <b>Vue CLI 4.0.4</b>
4. <b>Bootstrap 4.3.1</b>

## Project setup
```
npm install
```
```
npm run serve
```

For Login Page:
1. Username: string
2. Password: string
